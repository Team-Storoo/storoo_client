# 소셜 로그인 아이콘 폴더

여기에 아이콘 이미지 파일을 넣어주세요.

| 파일명       | 용도           |
|------------|--------------|
| kakao.png  | 카카오 로그인 버튼 아이콘 |
| naver.png  | 네이버 로그인 버튼 아이콘 |
| google.png | 구글 로그인 버튼 아이콘  |

- PNG 권장 크기: 48×48px 이상 (2x, 3x 대응 시 각각 `2.0x/`, `3.0x/` 폴더 활용)
- SVG를 사용하려면 pubspec.yaml에 `flutter_svg` 패키지를 추가하고 Image.asset → SvgPicture.asset으로 교체하세요.
